# ☕ Café Aroma – Coffee Shop Website

Café Aroma is a modern and responsive coffee shop website developed using HTML, CSS, and JavaScript.  
The project was created as part of an AWS Cloud website deployment assignment and is hosted using Amazon S3 Static Website Hosting.
